# 🚀 خطوات تفعيل الموقع من GitHub Pages

## ✅ **تفعيل GitHub Pages - خطوات سريعة**

### 📋 **الخطوات بالترتيب (نفذها الآن في دقيقة):**

#### 1️⃣ **ادخل على مستودعك:**
```
https://github.com/Codeplus3/buytuk31
```

#### 2️⃣ **اذهب إلى الإعدادات:**
- اضغط على التبويب **Settings** (الإعدادات) في الأعلى

#### 3️⃣ **اختر Pages:**
- من القائمة اليسرى، اختر **Pages**

#### 4️⃣ **إعدادات Build and deployment:**
- **Source**: اختر **Deploy from a branch**
- **Branch**: اختر **main**
- **Folder**: اختر **/docs** (هذا هو السر لأن ملفاتك هناك)

#### 5️⃣ **احفظ الإعدادات:**
- اضغط على زر **Save**

## 🎯 **لماذا هذه الإعدادات صحيحة:**

### ✅ **Source: Deploy from a branch**
- **النشر من فرع** - يستخدم فرع main تلقائياً
- **تحديث تلقائي** - كل push لـ main يحدث الموقع
- **بسيط وموثوق** - لا يحتاج إعدادات معقدة

### ✅ **Branch: main**
- **الفرع الرئيسي** - يحتوي على أحدث الكود
- **مستقر** - جميع التغييرات تم اختبارها
- **صحيح** - متوافق مع push الذي قمت به

### ✅ **Folder: /docs**
- **المجلد الصحيح** - يحتوي على موقع Jekyll كامل
- **السر الرئيسي** - جميع ملفات الموقع موجودة هنا
- **متوافق** - نفس الهيكل المستخدم في Cloudflare Pages
